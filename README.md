# iudizm.github.io

## Para rodar

To start the container simply run:

```sh
docker-compose up
```

Alternatively you can run the container without docker-compose using the following command on Mac/Linux:

```sh
docker run -p 4000:4000 -v $(pwd):/site bretfisher/jekyll-serve
```
